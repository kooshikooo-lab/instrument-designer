## Saxophone Key Mechanism

### Basic Boehm System Layout

The saxophone uses a conical Boehm-derived key system. The tone holes are arranged in a graduated conical bore with keys closing padded tone holes.

| Label | Finger | Mechanism | Function |
|-------|--------|-----------|----------|
| **LH Thumb** | L thumb | Octave key (register vent) | Two vents: neck vent (D4-F#4) and body vent (G4+). Pressing opens appropriate vent. |
| **LH1** | L index | Main hole + linked C# vent | Closes hole 1. When open, also opens linked C# auxiliary vent. |
| **LH2** | L middle | Main hole + linked Bb ring | Closes hole 2. Bb ring opens when LH2 lifts. |
| **LH3** | L ring | Main hole + linked A ring | Closes hole 3. A ring opens when LH3 lifts. |
| **LH4 (G#/C#)** | L pinky | Two side keys: G# (upper), C# (lower) | G# key opens G# pad. C# key opens C# vent on body. |
| **RH1** | R index | Main hole | Closes hole 4. |
| **RH2** | R middle | Main hole + linked F# ring | Closes hole 5. F# ring opens when RH2 lifts. |
| **RH3** | R ring | Main hole + linked F+E | Closes hole 6. Linked to E pad on bell. |
| **RH4 (low C#/Eb)** | R pinky | Eb/D# table key | Opens Eb/D# side vent on lower body. |
| **RH pinky (bell)** | R pinky | 3 bell keys: C / B / Bb | Each opens a pad on the bell flare. Normally closed, press to open. |

### Octave Key System

The saxophone has two octave vents (register holes) operated by a single LH thumb key:

| Register | Written Range | Vent Used | Action |
|----------|--------------|-----------|--------|
| **Low (fundamental)** | Bb3 to C#4 | None | LH thumb **not pressed** (вЂ”). The instrument plays in its fundamental register. |
| **Middle** | D4 to F#4 | Neck octave vent | LH thumb **pressed** (в—Џ). A small vent near the neck opens. |
| **Upper** | G4 to F#6 | Body octave vent | LH thumb **pressed** (в—Џ). A vent on the upper body opens; the neck vent closes automatically via a mechanical linkage. |

The body vent replaces the neck vent at written G4 because the neck vent would be at a pressure node (ineffective) for the G4+ partials.

### Fingering Column Conventions

Each row uses:

| Column | Convention |
|--------|-----------|
| **LH Thumb** | В«вЂ”В» = octave closed, В«в—ЏВ» = octave open |
| **LH1, LH2, LH3** | В«в—ЏВ» = key pressed (hole closed), В«в—‹В» = key open (hole open) |
| **LH4 (G#)** | В«в—‹В» = nothing pressed, В«в—Џ G#В» = G# key, В«в—Џ C#В» = C# key |
| **RH1, RH2, RH3** | В«в—ЏВ» = key pressed (hole closed), В«в—‹В» = key open (hole open) |
| **RH4 (low C#)** | В«в—‹В» = not pressed, В«в—ЏВ» = Eb/C# key pressed |
| **RH pinky** | В«в—‹В» = none pressed, В«в—Џ EbВ» = Eb key, В«в—Џ CВ» = C bell, В«в—Џ BВ» = B bell, В«в—Џ BbВ» = Bb bell |

**Note on notation**: For bell keys, В«в—ЏВ» means the key is **pressed** (which **opens** the pad, allowing air to escape through that vent). This is the standard fingering-chart convention.

### How RH4 / RH Pinky Interact

On the Boehm saxophone, the right pinky operates a 4-key table:

1. **Eb/D# key** (topmost) вЂ” opens an auxiliary vent on the lower body
2. **C key** вЂ” opens a pad on the bell
3. **B key** вЂ” opens a pad lower on the bell
4. **Bb key** вЂ” opens the lowest pad on the bell (extends the bell)

In our column model:
- **RH4 (low C#)** captures the Eb/D# auxiliary vent (used for Eb/D# and for stabilizing low notes)
- **RH pinky (Eb/Bb/C)** captures the three bell keys

For low register notes (Bb3вЂ“C#4), RH4 is В«в—ЏВ» (low mechanism engaged). For most upper notes, RH4 is В«в—‹В».

## Soprano vs Alto вЂ” Key Mechanism Differences

| Feature | Soprano Sax | Alto Sax | Impact |
|---------|-------------|----------|--------|
| **Bore shape** | Narrower conical bore, ~21вЂ“22mm at mpc end | Wider conical bore, ~24вЂ“25mm at mpc end | Smaller holes on soprano; same fingering geometry scaled |
| **Tone hole spacing** | Closer together (shorter tube, ~700mm) | Wider apart (longer tube, ~800mm) | Soprano has tighter finger spacing |
| **Bell** | Flare begins ~530mm from mpc; ~80вЂ“90mm bell opening | Flare begins ~650mm from mpc; ~120вЂ“140mm bell opening | Soprano bell is more directional |
| **Neck** | Straight neck, octave vent on body near top | Curved neck, octave vent on neck | Neck vent position differs but function same |
| **Low Bb key** | Present on most sopranos | Present on all altos | Both have full low range |
| **High F# key** | Standard on modern sopranos | Standard on modern altos | Both support front F altissimo |
| **Front F key** | Standard (auxiliary high F) | Standard (auxiliary high F) | Same mechanism on both |
| **Side keys (C, Bb)** | Present, same layout | Present, same layout | Same relative positions |
| **Palm keys (D, Eb, F, F#)** | Positioned higher relative to LH | Same relative to LH | Same playing feel scaled |
| **Octave vent position** | Neck vent ~80mm from mpc; body vent ~160mm | Neck vent ~90mm from mpc; body vent ~200mm | Different absolute positions but same relative |
| **Keywork** | More compact, lighter springs | Larger key spans, heavier springs | Soprano action is lighter/faster |

**Critical takeaway**: The **fingering system is identical** between soprano and alto. A player can switch between them without re-learning fingerings. The only differences are:
1. Transposition (M2 vs M6 below written)
2. Physical scale (smaller vs larger instrument)
3. Slight embouchure and voicing adjustments (soprano is more sensitive)

For TMM modeling, this means:
- The same hole-opening pattern maps to different sounding pitches
- Soprano frequencies are ~1.5Г— higher than alto for the same written fingering
- Hole position scaling follows the bore length ratio (alto ~1.2Г— longer than soprano)

## TMM Modeling Considerations

### Register Vent Modeling

The saxophone octave key system requires two register vents in the TMM:

| Vent | Purpose | Active Range | TMM Position |
|------|---------|-------------|--------------|
| Neck vent (upstream) | Overblow D4вЂ“F#4 | Written D4 to F#4 | ~40вЂ“90mm from mpc (varies by size) |
| Body vent (downstream) | Overblow G4+ | Written G4 to F#6 | ~130вЂ“200mm from mpc (varies by size) |

For the TMM model with `n_register=3` (2nd harmonic = octave), the register vent must be open for **all** target notes. This matches the physical instrument where the octave key is always on for the second register.

### Phase Convention

Per the existing model in `tmm-model.md`:

```python
def pipe_reply_phase(phase_end: float, length_on_wavelength: float) -> float:
    return phase_end + length_on_wavelength * 2.0
```

With `closed_top=False` (open-open conical bore), the total phase is:

```
phase = 1 + 2L/О»
Resonance: 1 + 2L/О» = n
f = c(n-1)/(2L)
```

For the 2nd register (octave key): **n_register = 3**.

### Fingering-to-Hole Mapping

For TMM optimization, the 6 main tone holes map to the fingerings as follows:

| Hole index | Closest to mpc | Controlled by | Opens when |
|------------|---------------|---------------|------------|
| 1 (highest) | H1 | LH1 | LH1 is в—‹ |
| 2 | H2 | LH2 | LH2 is в—‹ |
| 3 | H3 | LH3 | LH3 is в—‹ |
| 4 | H4 | RH1 or side Bb | RH1 is в—‹ (or side Bb pressed for some notes) |
| 5 | H5 | RH2 or side C | RH2 is в—‹ |
| 6 (lowest) | H6 | RH3 + LH4(C#) | RH3 is в—‹ (C# vent also opens for some notes) |

**Critical for optimization**: The cumulative fingering pattern (used in the existing 6-hole model) applies only to the **second register** (octave key on, written D5вЂ“B5 range on alto for the optimization targets). The fingerings in the charts above show that this pattern holds for notes D5 through B5 (written) in the second register.

### Notes Requiring Cross-Fingerings

The following written notes use **non-cumulative** fingerings (holes do not open sequentially from the bell):

| Written Note | Fingering Pattern | Issue for TMM |
|-------------|-------------------|---------------|
| Eb4 | All closed + Eb side key (LH3 can be open) | Uses auxiliary key, not a main hole |
| F4 | LH2 open only | Cross-fingering (fork F) |
| F#4 | LH2+LH3 open | Cross-fingering (fork F#) |
| G#4 | G + LH4 G# key | Uses LH4 auxiliary key |
| A#4/Bb4 | LH2 open only (bis) | Cross-fingering (uses Bb mechanism) |
| Eb5 | RH3 open + Eb side key | Uses auxiliary RH4 key |
| Altissimo (G#5+) | Various | Multiple auxiliary keys, highly fingering-dependent |

These notes will require special handling in the optimization вЂ” either including the auxiliary keys as additional holes or accepting larger tuning deviations.

### Acoustic Differences: Soprano vs Alto for TMM

| Parameter | Soprano | Alto |
|-----------|---------|------|
| Bore length (approx) | ~700mm | ~800mm |
| Mouthpiece-end diameter | ~11вЂ“12mm | ~13вЂ“14mm |

- Higher impedance peaks (tighter bore)

## Summary of Fingering Rules for TMM Input

1. **LH Thumb** = В«вЂ”В» for written Bb3вЂ“C#4 (fundamental register); В«в—ЏВ» for D4+ (octave register). For n_register=3 targets, always В«в—ЏВ».

2. **6 main tone holes** are controlled by LH1вЂ“LH3 and RH1вЂ“RH3. The cumulative opening pattern (opening from the bell end) works for notes in the second register D5вЂ“B5 (written).

3. **LH4 (G#/C#)** is an auxiliary key system with two pads. G# opens for G#/Ab, A, Bb, B. C# opens for C# notes and for stabilizing low B/Bb.

4. **RH4 (low C#)** operates the Eb/D# side vent. Used for Eb notes and low register stabilization.

5. **RH pinky** operates the bell keys (C, B, Bb). Only used for the lowest 4 notes (written Bb3вЂ“C#4) and for Eb/D# fingerings.

6. **Cross-fingerings** (F4, F#4, Bb4, G#4, Eb4, C#5) introduce non-cumulative hole states that require separate TMM optimization or higher tolerance.

7. **Octave vents** are not part of the fingering holes but are always open for the optimization targets (n_register=3). Their position and size affect harmonicity and should be modeled separately.

8. **Soprano and alto** share identical fingering patterns. Only the transposition (M2 vs M6) and the absolute frequency targets differ.
