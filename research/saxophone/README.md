# Saxophone Design Research

Research findings and literature review for computational saxophone design using TMM (Transfer Matrix Method).

## Key Papers

1. **Lefebvre & Scavone (2011)** — "On the bore shape of conical instruments"
2. **Szwarcberg (2025)** — "Geometric sensitivity of modal parameters in wind instrument models"
3. **Lefebvre PhD thesis (2010)** — "Computational Acoustic Methods for the Design of Woodwind Instruments"
4. **DAGA 2021** — "Tone Hole Adjustment for Wooden Saxophone Using an Open-Source 1D Waveguide"
5. **Noreland** — Tone hole optimization for clarinets and saxophones
